Redux React Router Example
=====================

* hot reloading React components;
* routing with redux-router and react-router;
* redux dev tools
* babel-plugin-react-transform
* webpack

## Run Dev

```
npm install
npm start
open http://localhost:3000
```

## Run Prod

```
npm install
npm run build
npm start:prod
open http://localhost:3000
```
